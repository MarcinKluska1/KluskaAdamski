## Install requirements
1. [Install Docker](https://docs.docker.com/install/linux/docker-ce/ubuntu/)

2. Get our project. 
```Shell
git clone https://github.com/MarcinKluska1/KluskaAdamski.git
```

3. Build docker image

4. Run docker session
